## Requirements

In order to use this program you need to download:

- Java 
- Gradle

## How to Build?

Using gradle in the project:

`./gradlew build`

(You might have to run the following command if the above does not work:
`gradle wrapper`)

## How to run?

Once you have built the project you can it:

`./gradlew run`

## How to test?

Using gradle:

`./gradlew test`
